produtividade-eclipse
===============

Aplicativo desktop via linha de comando (console). Esse projeto tem como objetivo complementar o estudo das funcionalidades do Eclipse IDE.

Vídeo-aula
-------
Ganhe agilidade ao trabalhar com o Eclipse. Assista aos screencasts (vídeo-aula) que demonstram dicas e macetes do Eclipse:
http://www.yaw.com.br/screencast

Pré-requisitos
-------
* JDK - última versão do Kit de desenvolvimento Java;
* Eclipse IDE
